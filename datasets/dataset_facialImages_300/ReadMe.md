# Facial Images Dataset 300 x 300 eyes Open and Close Eyes

## Source  
This dataset is taken from   
http://parnec.nuaa.edu.cn/xtan/data/TSKinFace.html  
The original Datset was of dimensions 100 x 100 but it is enhanced to 300 x 300 using SuperResolution https://github.com/pytorch/examples/tree/master/super_resolution

## Details
**Resolution:**  
    300 x 300 px

**Folder Structure:**
```
├───dataset_facialImages_300
    ├───test
    │   ├───CloseFace  # 179 images 
    │   └───OpenFace   # 182 images
    └───train
        ├───ClosedFace  # 1013 images
        └───OpenFace  # 1015 images
```
