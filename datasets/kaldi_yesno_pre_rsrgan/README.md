# KALDI yesno Dataset.

This DataSet is taken from [KALDI](https://github.com/kaldi-asr/kaldi) yesno dataset. And it is preprocessed according to an OpenSource Github Repository named [rsrGAN](https://github.com/wangkenpu/rsrgan).

It include audio files containing spoken words i.e. yes, no in .wav audios form and and its corresponding transcript files its all rights are reserve to KALDI. And it is preprocessed according to rsrGAN using KALDI's tools. Preprocessing information can be taken by reading README note of rsrGAN repository.

Note: Please if you want to use this you can get more proper DataSet from KALDI OpenSource.
